# Projeto-Jogar-Dados

## Apêndice:

Projeto tem por objetivo simular o jogo de um dado 
de 06 faces.

## Requisitos:

- Interação do usuário para jogar o dado ao clicar no botão JOGAR.
- Ao clicar no botão retorna o resultado do valor do dado.
